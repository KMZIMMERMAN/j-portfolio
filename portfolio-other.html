About

A website for that visually displays Twitter account data and trends for @RobHasApodcast.

Background

This website uses Twitter's Public API and CSV file data to visually display account data and trends.

It uses front-end and back-end operations to upload, fetch, and store account information. API data updates automatically every 5 hours or via the "Refresh Data" button (due to API limits, the button can only be pressed a limited number of times). Data that requires CSV files will update when a CSV file is uploaded.

I used my Twitter account's CSV data (& randomized numbers) to showcase the file-uploading feature temporarily, it is not the RHAP account's data. I also plan to implement uploading the Audience CSV data, but my personal Twitter does not currently have the option.

Recent Tweet Analysis includes calculations for Brand Impact Score, Content Performance Score, Click-Through Rate, Call-to-Action (CTA), and more to determine the effectiveness of posts.
